# 🎙️ TEDMaster (TED 精读大师)

[![AI Powered](https://img.shields.io/badge/AI-Powered-7c3aed?style=for-the-badge&logoColor=white)](https://google.com/gemini)
[![Framework: Next.js](https://img.shields.io/badge/Framework-Next.js-000000?style=for-the-badge&logo=nextdotjs)](https://nextjs.org/)
[![Design: Glassmorphism](https://img.shields.io/badge/Design-Glassmorphism-FF4B4B?style=for-the-badge)](https://en.wikipedia.org/wiki/Glassmorphism)

**TEDMaster** 是一款专为英语进阶者打造的沉浸式精读站。它利用先进的 AI 技术，将 TED 演讲从视觉冲击转化为深度的语言习得体验。

---

## ✨ 核心特性

- 💎 **高级感毛玻璃界面**：深色模式下的沉浸式精读体验，无干扰。
- 🤖 **AI 驱动的即时反馈**：
  - **即点即查**：在字幕中点击任何单词，即刻获取当前语境下的精准定义。
  - **长难句拆解**：AI 自动解析语法结构，让每个从句无所遁形。
- 🔊 **影子练习 (Shadowing)**：内置音波对比功能，AI 实时评价你的发音与语调。
- 📝 **极简生词笔记录**：一键收藏带原句音频的生词卡片，支持 Anki 导出。
- 🔄 **TED 海量实时同步**：自动抓取 TED 官方多语言字幕，无需手动整理。

---

## 🛠️ 技术演进

- **Frontend**: Next.js 14, React, TypeScript
- **State**: React Server Components & Context API
- **AI Integration**: Google Gemini 1.5 Pro
- **Storage**: PostgreSQL + Redis (Caching)
- **Styling**: Vanilla CSS (CSS Variables)
